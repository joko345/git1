Javascript bersifat single threaded(call stack) atau syncronous

program dijalankan dari urutan atas ke bawah. dalam javascript asyncronous

disediakan oleh runtime environtment dalam browser yang menjalankan fungsi web api

terdapat event loop yang akan memasukkan javascript ke queque stack atau ansycronous

contoh: setTimeout adalah fungsi asyncronous, saat dijalankan akan masuk event loop, lalu ke callstack di callstack, javascript akan sadar itu fungsi runtime environtment, kemudia akan dimasukan ke web api setelah itu masuk ke queque stack. saat fungsi itu di queque maka callstack bisa menjalakan fungsi lain karena queque dijalankan sesuah callstack. saat callstack tidak ada fungsi lain, maka event loop mengecek queque dan memindahkannya ke callstack dan di dijalankan.
